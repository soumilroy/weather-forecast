# Weather forecast challenge

Codementor challenge using React, Chartjs & OpenWeather API

Use this project for yourself

1. Clone git repo
2. `cd weather-forecast/`
3. `yarn install`
4. Rename `.env-example` to `.env`
5. Paste API Key to .env file
6. `yarn start`
